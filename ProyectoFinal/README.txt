Para ejecutar el programa abriremos el fichero PruebaInterfaz.m en MATLAB y le damos a RUN.
A partir de ahí basta con seguir las instrucciones que se muestra en la interfaz para poder utilizar la aplicación.

Para poder continuar una partida ya empezada se guardará en la misma carpeta donde está el código una imagen de ella
en formato jpeg y de nombre "tablero". Se puede ver un ejemplo en la imagen que adjuntamos en la carpeta con dicho 
nombre y extensión.
